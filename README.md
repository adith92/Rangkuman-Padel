# Padel Sport Indonesia

Direktori nasional PBPI, lapangan padel, kontak olahraga, Rekor MURI, serta titik publik TNI AD, AL, dan AU.

## Menjalankan lokal

Buka `index.html` secara langsung. Untuk peta online, perangkat harus terhubung ke internet agar Leaflet dan tile OpenStreetMap dapat dimuat.

## Struktur

- `index.html` — antarmuka utama
- `styles.css` — tampilan responsif
- `data-1.js`, `data-2.js`, `data-3.js` — data titik peta
- `app.js` — navigasi, filter, peta, dan geocoding
- `archive/` — versi HTML sumber

> Data berasal dari sumber publik dan wajib diverifikasi sebelum dipakai untuk surat resmi, kunjungan, atau keputusan operasional.
